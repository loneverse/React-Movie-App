### Installation

`git clone https://github.com/loneverse/React-Movie-App.git`

### Running the Project
#### After cloning the Repo You need to Run:

`npm i react-scripts `
- To set up react-scripts

`rm -rf node_modules`
- To remove the node modules

`npm install`
- download all deps 

`npm start`
- Start your project
